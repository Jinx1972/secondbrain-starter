# 05 Daily Notes

Das Logbuch, eine Datei pro Tag im Format `2026-07-21.md`.

Am Ende eines Arbeitstags genügt der Satz „Fass den Tag zusammen". Claude legt die Notiz an und trägt ein, was passiert ist und was offen bleibt. Beim nächsten Mal weiß er dadurch, wo ihr stehen geblieben seid.

Diese Notiz kannst du löschen.
